Extension: no-basis-propertyinformation

This basis extension describes information identifying norwegian real property.

The official URL for this extension is: 

http://hl7.no/fhir/StructureDefinition/no-basis-propertyinformation

Status: active
Extension maintained by:

Context of Use

This extension may be used on the following element(s):

  • Address

Usage info

Usage:

  • Use this Extension: no-basis-Address
  • Examples for this Extension: Patient/EspenEksempel, Patient/JanniceSoreng, Patient/JanniceSorengTo, RelatedPerson/Solid-Aresdoktor-RelatedPerson
